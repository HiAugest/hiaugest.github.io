Java8 Lambda表达式

2019年9月26日
| java基础
Lambda使用前提 在java8的新特性中，接口中的方法可以有方法主体，但必须要使用static修饰或default修饰。由此衍生出了一个新的规则：函数式接口 函数式接口： 当接口中有且只有一个抽象方法，但可以有其他的方法 注解：@FunctionInterface 表示该接口为函数式接口，只能由一个抽象方法 Lambda表达式 1.作用 1.简化代码，使代码异常简洁：可以非常优雅的把“一块代码”赋给了一个变量 2.相当于匿名内部类的简写，但不会生成字节码文件 2语法结构 接口名 变量名 = （参数类型 参数名） -> {功能语句};……
